## 에러 자동 기록

사용자가 Claude의 실수를 수정하거나 지적하면, 아래 `## Error Log` 섹션에 자동으로 기록할 것.

**기록 형식:**
```
- [YYYY-MM-DD] [프레임워크/영역] 잘못한 내용 → 올바른 방식
```

**기록 기준:**
- 사용자가 명시적으로 수정한 경우
- 같은 실수를 2회 이상 반복한 경우
- 프로젝트 컨벤션을 어긴 경우

**기록 금지:**
- 사용자의 요구사항 변경 (실수가 아님)
- 일회성 오타
